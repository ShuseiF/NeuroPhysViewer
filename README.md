# NeuroPhysViewer

NeuroPhysViewer is an open-source R Shiny application for visualizing and analyzing extracellular electrophysiological recordings exported from LabChart.

## Current Features

- LabChart TXT import
- Interactive waveform viewer
- Synchronized Unit / MT / TTL display
- Plotly zoom and pan

## Planned Features

- Performance optimization
- TTL event detection
- LabChart label visualization
- Spike detection
- Raster plot
- PSTH
- Batch analysis
- ADInstruments (.adicht) support
